# ZAS Safeguard for OpenClaw

Control your browser security and parental controls from Telegram or WhatsApp.

## What it does

- Start and stop Focus Mode to block distractions
- Enable Internet Lock to block everything except essential sites
- Scan any URL for malware, phishing, and adult content
- Check your protection stats and recent blocked sites
- Manage your custom blocklist
- Monitor all connected devices

## Setup

1. Install the skill: `claw install zas-safeguard`
2. Connect your account: `zas:connect your@email.com yourpassword`
3. Done — start with `zas:status`

## Examples

> "Start focus mode for 2 hours"  → `zas:focus start 2h`
> "Is this link safe? https://example.com"  → `zas:scan https://example.com`
> "Lock the internet, kids are going to bed"  → `zas:lock`
> "What did ZAS block today?"  → `zas:activity`
> "Add reddit.com to my blocklist"  → `zas:block add reddit.com`

## Requirements

ZAS Safeguard account (free or Pro) — sign up at zassafeguard.com
Pro subscription required for Focus Mode, Internet Lock, and advanced stats.
